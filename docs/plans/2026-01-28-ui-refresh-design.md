# MBTI 在线测试网站 UI 重构设计（温暖疗愈 / 浅色为主 + 轻暗对比区）

日期：2026-01-28

## 目标

- 将现有页面从偏“深色霓虹”风格，调整为“温暖疗愈、浅色为主”的视觉。
- 允许少量“轻微暗色块/对比区”来提升层级与可读性，但不做完整暗黑模式。
- 覆盖全站：用户端（首页/答题/完成/结果）+ 后台管理页（题库列表/编辑/导入导出）。
- 保持 SSR + 轻 JS（可选复制按钮）思路，不引入前端框架。
- 优先可读性与可访问性：清晰的对比度、键盘 focus、减少动画模式。

## 风格关键词（ui-ux-pro-max）

- Soft UI Evolution（更好的对比、更现代的柔和阴影、可访问性优先）
- 轻量玻璃质感（浅色半透明卡片 + 柔和阴影）

## 颜色系统（选择：珊瑚柔雾）

基础色板（来自 ui-ux-pro-max 推荐并结合站点需求）：

- 主色（Primary）：`#EC4899`（珊瑚粉）
- 辅色（Secondary）：`#F9A8D4`（浅粉）
- 强调/CTA：`#8B5CF6`（薰衣草紫）
- 背景（Background）：`#FDF2F8`（柔雾粉白）
- 正文（Text）：`#831843`（深莓红）

轻暗对比区（用于提示、次级卡片底、表头等）：

- 暗色基底：`#3B0F21`
- 使用方式：以 `8–12%` 透明度叠加（例如 `rgba(59, 15, 33, 0.08)`），避免“黑块压迫感”。

语义色：

- 危险/退出：偏玫红（与主色同系），但降低饱和度与透明度，避免刺眼。
- 成功/完成：可选薄荷绿（仅用于小范围提示，不主导品牌）。

## 字体系统

- 标题：Varela Round（圆润亲和）
- 正文：Nunito Sans（易读、柔和）
- 回退：系统 sans-serif

说明：字体通过 Google Fonts 引入；在网络不可用时使用系统字体回退，保证可用性。

## 组件规范

### 布局

- 顶部导航：浅色磨砂（半透明 + blur），固定/粘性不挡内容。
- 内容容器：`max-width` 适度加宽（≈ 960px），移动端自动收缩。
- 背景：浅色底 + 轻渐变光斑（珊瑚粉/薰衣草紫/浅粉），增强“疗愈感”。

### 卡片（Card）

- 统一圆角（≈ 16–18px）
- 白底/浅粉半透明，边框淡化，阴影柔和扩散
- 支持“对比卡”（轻暗底）用于提示/汇总

### 按钮（Button）

- 主按钮：粉→紫渐变 + 白字（用于“开始测试/下一题/生成报告”）
- 次按钮：浅底描边（用于“上一题/返回”）
- 危险按钮：浅玫红底 + 深色文字（用于“保存并退出”）
- 动效：150–250ms 的颜色/阴影过渡；支持 `prefers-reduced-motion` 降低动画

### 表单与选择题

- 模式选择：使用“卡片式单选”（20/40/60 三档），更直观。
- 量表选项：整行可点的软卡（label 包裹 radio），选中态清晰（描边 + 轻底色）。
- 输入框：浅底、边框对比足够，focus ring 明显。

### 进度与结果图表

- 答题进度条：细条 + 当前题号徽章，突出“当前进度”。
- 百分比条：统一圆角条，主色渐变填充，旁边显示百分比与差距。

### 后台表格

- 表头使用轻暗对比底提升可读性
- 行 hover 明显但不跳动
- 操作区（新增/导入/导出）与筛选区分层

## 可访问性与质量清单

- 文字对比度符合 WCAG AA（尤其浅色背景上的正文/弱化文字）。
- 所有可交互元素有 `cursor: pointer`、hover、focus-visible。
- 表单控件有 label；radio 的 label 可点区域足够大。
- 兼容移动端（375px）到桌面（1440px），不产生横向滚动。
- 尊重 `prefers-reduced-motion`：减少或关闭动画。
